Welcome to the Steam Database Project

Explore user profiles, game stats, and achievements.

Database Features

  • Relational Database: Built using an Sqlite3 database backend.
  • Entity Relationships: Efficient normalized organization and retrieval with minimized redundancy and optimized relationships.
  • Distilled Information: Stores a curated subset of data from Steam Web API.

Application Features

  • User Profiles: Track game achievements and view connected friends.
  • Game Stats: Compare player stats and track playtime.
  • Achievements: Discover unlocked in-game achievements.
